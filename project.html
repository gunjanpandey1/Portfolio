A * Path Finding Algorithm

Explore my project, Star Path Finder, a visually interactive tool that demonstrates the A* Pathfinding algorithm. This web-based application allows users to set start and end points, place obstacles on a grid, and watch the algorithm find the optimal path in real-time. Built using Python and Pygame, the project is designed to provide an educational and intuitive experience for understanding one of the most popular search algorithms used in computer science.
